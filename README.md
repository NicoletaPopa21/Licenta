### Dataset ECG Heartbeat Classification dataset
The project uses the ECG Heartbeat Classification dataset (Kachuee et al., 2018), available on arXiv:
https://arxiv.org/abs/1805.00794

The dataset contains segmented ECG heartbeat signals used for arrhythmia classification.

### Dataset Brain tumor multimodal image (CT & MRI)
This project uses the following dataset:
Murtozalikhon. (2023). Brain Tumor Multimodal Image (CT and MRI). Kaggle.
